## Design System

### Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Primary | #3B82F6 | Buttons, links, active states |
| Primary Dark | #2563EB | Hover states |
| Success/Normal | #10B981 | Normal vitals, success states |
| Warning | #F59E0B | Warning alerts, caution states |
| Critical/Error | #EF4444 | Critical alerts, errors |

| Background Dark | #1F2937 | Dark mode background |

## UI/UX Requirements

### Accessibility (WCAG AA)
- High contrast for readability
- Minimum touch target: 44x44px
- Screen reader support
- Color-blind friendly (don't rely on color alone)
- Text scalable to 200%

### Responsive Design
- Desktop-first (caregiver stations)
- Tablet optimized (iPad)
- Mobile optimized (iPhone)
- Minimum width: 320px

### Performance
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3s
- Smooth scrolling (60fps)
- Lazy load images
- Skeleton loaders for data

### Dark Mode
- System preference detection
- Manual toggle in settings
- All colors have dark variants
- Reduced brightness for night shift

### Animations
- Subtle transitions (200-300ms)
- Loading spinners for async operations
- Pulse animation for critical alerts
- Smooth chart updates

---

## Charts & Visualizations

### Vital Trend Line Chart
- Interactive zoom and pan
- Tooltip on hover/tap
- Multiple series overlaid
- Threshold lines shown
- Annotations for events

### Activity Bar Chart
- Daily steps comparison
- Week view default
- Tap for daily detail

### Sleep Graph
- Stacked area chart
- Deep/Light/REM/Awake
- Night-by-night comparison

### Alert Heatmap
- Hour-by-hour grid
- Color intensity = volume
- Tap to see details

### Map View
- Facility boundary shown
- Real-time resident markers
- Color-coded by status
- Tap for resident detail
